# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-04-26

### Added
- Initial release
- `mosaic` command: Convert images to emoji mosaics
- `banner` command: Generate emoji text banners
- CIEDE2000 perceptual color matching
- Multiple emoji palettes (apple, google, twitter, universal)
- PNG and text output formats
- Batch directory processing
- Clipboard copy support
